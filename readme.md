## Personality	
- 一组精神特质（类型）
    - 解释和预测思想，感觉和方式的模式 行为
    - 随着时间和环境保持相对稳定
- 人与人之间的差异来源
    - 影响关系，工作，学习等的成功概率
    - 解释35％的生活满意度差异
- 比较：收入（4％），就业（4％），婚姻状况（1％到4％）
- 检测与欺骗行为相互作用，意见，情感等

## Application
- 人格检测
- 精准广告
- 自适应接口和机器人行为
    - 音乐，风格，色调，色彩等
- 心理诊断，法医学
    - 精神病患者，大规模杀人犯，欺凌者，受害者
    - 抑郁，自杀倾向
- 人力资源管理
- 文学科学研究，社会心理学，社会语言学等
